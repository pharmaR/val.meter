# Metric data frame

A [`data.frame`](https://rdrr.io/r/base/data.frame.html), for all
intents and purposes, with column names corresponding to
[`pkg`](https://pharmar.github.io/val.meter/reference/pkg.md) metric
data.

## Usage

``` r
class_metric_data_frame(.data = list(), row.names = NULL)
```

## Arguments

- .data, row.names:

  See
  [`S7::class_data.frame`](https://rconsortium.github.io/S7/reference/base_s3_classes.html)
