# Parse an object from a `DCF`-formatted string

Parse an object from a `DCF`-formatted string

## Usage

``` r
from_dcf(x, to, ...)
```

## Arguments

- x:

  `character(n)` `DCF` vector

- to:

  A
  [`S7::S7_object`](https://rconsortium.github.io/S7/reference/S7_object.html)
  class to convert into

- ...:

  Additional arguments unused

## Value

An object of type `to`
