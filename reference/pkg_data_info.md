# Derive package data information

Retrieve metadata about a data field.

## Usage

``` r
pkg_data_info(field, ...)
```

## Arguments

- field:

  `character(1L)` field name to retrieve.

- ...:

  Additional arguments unused.

## Value

`data_info` for field `field`.
