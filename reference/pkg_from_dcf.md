# Produce `pkg`(s) from a `DCF`-formatted string

Produce `pkg`(s) from a `DCF`-formatted string

## Usage

``` r
pkg_from_dcf(x, ...)

pkgs_from_dcf(x, ...)
```

## Arguments

- x:

  A `DCF`-formatted string

- ...:

  Additional arguments unused

## Value

A `pkg` object. Note that when parsing from a `DCF` string.
