# val.meter: validation-ready package quantification supporting regulatory use of R

Measures packages for common best practices, adoption and code
execution. Built with regulated industries in mind, supporting the
surveying of packages and aggregating findings into a portable format to
support activities like selecting mature packages and reporting on
risk-based software decision-making.
